```

├── index.html          ← Main portfolio page

│   └── main.js         ← Animations & interactivity
└── README.md
```

## 🚀 How to Use

1. **Open locally**: Simply double-click `index.html` in your browser.
2. **Host online**: Upload all files to any web host:
   - [GitHub Pages](https://pages.github.com) (free)
   - [Netlify](https://netlify.com) (drag & drop, free)
   - [Vercel](https://vercel.com) (free)

## 🌐 Hosting on GitHub Pages (Recommended — Free)

1. Create a new GitHub repo named `spoorthika1.github.io`
2. Upload all files from this folder
3. Go to **Settings → Pages → Deploy from branch → main**
4. Your portfolio will be live at `https://spoorthika1.github.io`

## ✏️ Customizing

- **Update links**: Search for `spoorthika1` and `spoorthika-p-a5b5972a9` in `index.html` to update GitHub/LinkedIn URLs.
- **Add projects**: Copy a `.project-card` block in `index.html` and edit the content.
- **Colors**: Edit the CSS variables at the top of `css/style.css`.
- **Profile photo**: Replace the avatar initials section with an `<img>` tag pointing to your photo.
